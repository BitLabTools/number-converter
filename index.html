About

This converter keeps Decimal, Hexadecimal, Binary, and DNA code in sync for any signed integer size using JavaScript BigInt arithmetic. DNA code maps 2-bit chunks to bases: A=00, T=01, C=10, G=11.

  • Hex case is toggleable (A–F vs. a–f).
  • Binary has no leading zeros (except zero itself).
  • When deriving DNA from binary, if the bit-length is odd, the binary string is left-padded with one leading 0 so 2-bit chunks can be formed.
  • Editing DNA rebuilds the value via its 2-bit encoding.
